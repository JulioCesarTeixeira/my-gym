My Gym is a companion mobile application for fitness enthusiasts.
Built with React Native and TypeScript, the app offers a comprehensive exercise directory that allows users to keep track of their workout routines and fitness progress.

## Table of Contents

1. [Features](#features)
2. [Technologies](#technologies)
3. [Setup](#setup)

## Features

- User registration and authentication.
- Viewing exercises, filtered by muscle groups.
- Detailed view of each exercise, complete with an execution GIF.
- Completion of exercises that will automatically add to user's exercise history.
- Exercise history tracked and grouped by day.
- Profile customization with avatar image upload and password change functionality.

## Technologies

- **React Native w/ Expo**: The framework used for building the application.
- **TypeScript**: Applied to enhance application reliability and developer productivity.
- **React Query**: Used for data fetching, caching, and server state updates.
- **Async Storage**: Helps to persist user sessions across app launches.
- **Context API**: Utilized for managing global state.
